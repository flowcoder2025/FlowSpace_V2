# 치비 스프라이트 생성 품질 체크리스트

## 매 생성 후 반드시 검증 (3방향 모두)

### 1. 방향 정확도
- [ ] front: 정면 직시, 얼굴 완전 노출
- [ ] back: 완전 후면, 얼굴 없음
- [ ] left: 좌측 프로파일, 코/눈 측면

### 2. 캐릭터 일관성 (3방향 비교)
- [ ] 머리색: 동일 톤 (front/back/left)
- [ ] 머리 스타일: 동일 길이감/실루엣
- [ ] 의상 색상: 셔츠, 바지 동일 색상
- [ ] 체형 비율: 비슷한 너비/높이
- [ ] 성별: 동일

### 3. 스타일 통일
- [ ] 아웃라인 두께: 3방향 동일
- [ ] 채색 스타일: 동일 (셀셰이딩 등)

### 4. 구조적 완성도
- [ ] 전신: 머리~발끝까지 잘리지 않음
- [ ] 팔: 3방향 모두 팔 보임
- [ ] 불필요한 요소 없음 (주황 포켓, 밴드 등)

### 5. 배경 제거 (Rembg)
- [ ] 캐릭터 에지 깨끗함 (잘려나간 부분 없음)
- [ ] 투명 배경 정상
- [ ] 캐릭터 내부에 투명 구멍 없음

### 6. 파이프라인 일관성
- [ ] IP-Adapter ref는 Rembg 전 원본 사용
- [ ] 프롬프트에 캐릭터 설명(머리색, 의상) 일치
- [ ] seed 고정 확인

## 규칙
- Rembg는 최종 저장 직전에만 적용 (ref 이미지에는 적용 금지)
- IP-Adapter ref는 배경 있는 원본 사용
- 체크리스트 FAIL 항목이 있으면 "성공"으로 보고 금지
