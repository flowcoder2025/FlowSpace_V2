# 애니메/치비 머리스타일 정면→후면 변환 가이드

> 정면 이미지를 보고 후면이 어떻게 생겨야 하는지 판단하기 위한 참고 문서.
> QC 체크 시 반드시 참조.
> **Created: 2026-02-25**

---

## 일반 원칙

### 구조적 규칙
- 머리카락 3구역: **앞머리(bangs)**, **옆머리(sidelocks)**, **뒷머리(back)**
- 후면에서는 뒷머리 + 옆머리 바깥쪽만 보임
- 뒷머리는 두개골 형태를 따라 내려가며 중력 방향으로 흐름
- 머리카락 다발(clump) 사이에 fold lines 필요 — 없으면 2D 판지처럼 보임

### 치비 특수 규칙 (2~3등신)
- 머리가 과대 → 뒷머리 볼륨도 큼 (뒤통수가 큰 구 형태)
- 목이 거의 안 보임 → nape(뒷목) 표현 최소화
- **실루엣이 핵심** — 작은 스프라이트에서 뒷모습만으로 캐릭터 식별 가능해야 함
- 머리카락 다발 수 축소 (2~4개), 끝이 부드럽게 둥근 형태

### AI 생성 공통 문제
1. **Ghost Face**: 뒷통수에 눈/코/입 생성 (최대 문제)
2. **머리카락-의상 용해**: 아래쪽 머리카락이 의상과 합쳐짐
3. **앞머리 길이 불일치**: 정면보다 짧거나 긴 뒷머리
4. **색상 변이**: IP-Adapter ref의 색감이 후면에서 바뀜 (특히 금발/밝은색)

## 유형별 후면 가이드

### 1. 짧은 남성 머리, 옆가르마 (c01 정장)
- **후면 특징**: 뒷목(nape) 완전 노출, 머리카락이 두개골에 밀착, 볼륨 적음
- **가르마**: 후면에서 거의 안 보임 (정면 특징)
- **귀**: 끝부분 살짝 보임
- **치비**: 부드러운 둥근 캡 형태, 아래쪽에 작은 다발 2~3개
- **주의**: 후면에서 가르마가 보이면 안 됨, 앞머리 같은 디테일 금지

### 2. 긴 직모 (c02 검은 긴머리)
- **후면 특징**: 등까지 내려오는 거대한 머리카락 커튼, 세로 fold lines 필수
- **어깨 상호작용**: 어깨에서 펼쳐지며 넓어짐
- **뒷목**: 완전히 가려짐
- **옆머리**: 어깨 앞으로 살짝 보일 수 있음
- **치비**: 몸 아래까지 내려오는 큰 덩어리 1~2개, 끝이 부드러움
- **주의**: 정면의 길이와 반드시 일치, ghost face 위험 높음

### 3. 헝클어진/뾰족한 짧은 머리 (c03 후드)
- **후면 특징**: 정수리에서 방사형으로 뾰족한 다발, 정면과 **방향이 다름** (앞은 위/앞, 뒤는 바깥/아래)
- **뒷목**: 다발 사이로 부분 노출
- **치비**: 4~6개 큰 뾰족 다발이 방사형
- **주의**: 정면 스파이크를 그대로 복사하면 안 됨 — 뒤쪽 스파이크는 방향이 다름

### 4. 짧은 단발 (c04 스웨터)
- **후면 특징**: 아래쪽에 **수평 일직선 커트라인** (단발의 핵심), 끝이 안쪽으로 살짝 굽음
- **뒷목**: 커트라인 아래로 노출 (길이에 따라)
- **귀**: 끝부분 보임
- **치비**: 헬멧 같은 둥근 캡 + 깔끔한 아래쪽 직선, 짧은 남성 머리보다 볼륨 약간 더 큼
- **주의**: 후면에서 너무 길어지는 오류, 웨이브 추가 오류

### 5. 짧고 단정한 백발 (c05 조끼)
- **후면 특징**: 두개골에 밀착, 볼륨 매우 적음, 뒷목 완전 노출, 깔끔한 hairline
- **색상**: 흰색/회색이 주요 식별자 (형태는 다른 짧은 머리와 유사)
- **치비**: 두개골보다 약간만 큰 매끈한 둥근 캡
- **주의**: **볼륨 과다 = 불일치** — 부풀려진 둥근 백발 덩어리는 틀림. 단정하고 평평해야 함
- **AI 최대 위험**: 색상 변이 (정면 순백 → 후면 회색), 볼륨 과다

### 6. 짧은 금발 (c06 티셔츠)
- **후면 특징**: c01과 유사하나 약간 캐주얼 (1~2개 살짝 튀어나온 다발)
- **뒷목**: 노출
- **치비**: 둥근 캡 + 작은 튀어나온 다발로 캐주얼 느낌
- **주의**: **금발 색상 불안정** — Animagine XL에서 golden/platinum/brown 사이 흔들림. 텍스처 소실 시 대머리처럼 보임

### 7. 긴 머리 + 리본 (c07 교복)
- **후면 특징**: **리본/보우가 주요 랜드마크** — 보우 루프 + 리본 꼬리가 가장 눈에 띄는 요소
- **리본 위치**: 정면과 같은 높이·위치에서 후면으로 보여야 함
- **리본 아래 머리**: 자유롭게 아래로 흐름
- **치비**: 과대한 리본(캐릭터 식별자) + 아래쪽 큰 머리 다발 1~2개
- **주의**: **리본 실종** (AI가 정면 구도와만 연관지어 후면에서 생성 안 함), 리본 위치 변동, 이중 리본

### 8. 포니테일/올림머리 (c08 카페)
- **후면 특징**: **머리끈 + 꼬리 다발** — 후면에서 가장 눈에 띄는 요소
- **두개골 위쪽**: 매끈하게 당겨진 머리카락 (tension)
- **꼬리 형태**: "물음표" 형태 — 바깥으로 굽었다가 아래로 떨어짐
- **뒷목**: 높은 포니테일이면 노출, 낮으면 가려짐
- **치비**: 둥근 머리 캡 + 뚜렷한 꼬리 하나가 실루엣 식별자
- **주의**: 꼬리가 정중앙이면 혹처럼 보임 (약간 한쪽으로 치우쳐야), 머리끈 누락

### 9. 짧은 검은머리 + 셰프모자 (c09)
- **후면 특징**: **원통형 하얀 토크** 가 시야 지배, 주름선(pleats) 보임
- **머리카락**: 대부분 모자 안에 숨김, 뒷목/옆에서 살짝만 보임
- **모자 밴드**: 수평 라인
- **치비**: 과대한 하얀 원통 + 머리는 거의 안 보임
- **주의**: **모자 형태 변형** (원통→베레/플랫), 뒤에서 모자가 제대로 입체감 있어야 함

### 10. 짧은 머리 + 갑옷 (c10 기사)
- **후면 특징**: 헬멧 뒷판 + 목가리개(neck guard), 머리카락 거의 안 보임
- **망토**: 어깨에서 부착, 후면에서 잘 보임
- **치비**: 과대한 헬멧 + 망토가 실루엣 식별자
- **주의**: **정면 디테일(바이저 구멍 등)이 후면에 나타나면 안 됨** — 후면은 매끈한 뒷판

### 11. 매우 긴 머리 + 마법사 모자 (c11)
- **후면 특징**: 뾰족한 모자 원뿔 + 모자 챙(brim) 아래로 흘러내리는 긴 머리
- **모자 끝**: 한쪽으로 늘어짐 (뒤 또는 옆)
- **머리**: 모자 챙 아래에서 압축→자유 흐름으로 전환
- **치비**: 과대한 뾰족 모자 + 큰 머리 다발 = 매우 키 큰 실루엣
- **주의**: 모자 실종, 머리가 모자와 무관하게 흐름, 모자 형태 왜곡

## QC 체크리스트 (후면 머리스타일 전용)

1. **얼굴 특징 없음**: 눈/코/입, 얼굴 같은 패턴 없음
2. **머리 길이 일치**: 정면과 비교하여 몸 대비 길이 동일
3. **머리색 일치**: 정면과 색조 변이 없음 (특히 금발/백발)
4. **볼륨 일치**: 후면이 정면보다 극단적으로 크거나 작지 않음
5. **악세서리 존재**: 리본/모자/헬멧이 정면에 있으면 후면에도 반드시 있어야 함
6. **악세서리 형태 일치**: 정면과 같은 형태·위치 (모자 원통형, 리본 위치 등)
7. **적절한 뒷목 노출**: 짧은 머리=보임, 긴 머리=안 보임
8. **앞머리 안 보임**: 후면에서 bangs/sidelocks 보이면 안 됨
9. **머리 텍스처 존재**: 매끈한 대머리 같은 덩어리가 아닌 다발/fold 존재
10. **중력 방향 준수**: 머리카락이 아래로 흐름 (바람/동작 없을 때)
